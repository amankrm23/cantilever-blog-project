# Full-Stack Content Management System (Blog)

This project is a fully responsive, Full-Stack web application built for the Cantilever Labs internship evaluation. It features a complete C.R.U.D. architecture with a modern, fluid UI.

## 🚀 Advanced Features Built
* **Rich Text Editing:** Integrated Quill.js for WYSIWYG text formatting.
* **Media Uploads:** Configured Multer on the Express backend to handle physical `multipart/form-data` image uploads.
* **Live Search Engine:** Client-side filtering that instantly searches post titles and content without reloading the page.
* **Quick Index Directory:** A dynamic, horizontal UI pattern for rapid post navigation.
* **Responsive Fluid Design:** Custom CSS grid that adapts seamlessly from desktop to mobile screens.

## 💻 Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript, Quill.js
* **Backend:** Node.js, Express.js, Multer
* **Database:** MongoDB, Mongoose

## 🛠️ How to Run Locally
1. Ensure MongoDB Compass is installed and running on `localhost:27017`.
2. Clone this repository to your local machine.
3. Open the terminal in the project directory and run `npm install` to generate the `node_modules`.
4. Start the backend server by running `node server.js`.
5. Open `index.html` in your web browser.
